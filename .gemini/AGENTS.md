# Gemini Adapter

This file is a thin adapter for Gemini-compatible agents.

Primary contract:
- Read root `AGENTS.md` first.

Canonical content locations:
- Skills: `.claude/skills/`
- Agents: `.claude/agents/`
- Rules and hooks: `.claude/rules/`, `.claude/hooks/`

Usage policy:
- Treat `.gemini/` as adapter/bootstrap only.
- Do not duplicate canonical skill or agent content into `.gemini/`.
